HTML

  • The Head Element contains information about the page
  • The Body element repsesents the visible content shown to the user
  • The head element contains Metadata for the web page and inside contains tags
  • Title- very important for the SEO search engine optimization
  • Style- utalizing CSS for The look of the desired website
  • Base- this being the URL
  • Link- connects external sources to HTML doc
  • Meta- character set, description, keywords, author, viewpoint
  • Script- embeds client side to HTML doc
  • http-equiv- sets HTTP header for content
  • Viewpoint- sets default viewpoint for web page
  • HTML has SEMANTIC elemnts i.e.- opening/closing tags

CSS

  • A Margin indicates how much space we want around the OUTSIDE of the element
  • A Padding indicates how much space we want INSIDE the element
  • There are 3 way to use CSS STYLING: INLINE,INTERNAL, and EXTERNAL
  • INLINE- styles elememts directly effevt in the html doc.
  • -Quick permanent changes and could be applied to specific html tags
  • -BUT it is LESS flexible then INTERNAL and EXTERNAL
  • INTERNAL- contains CSS rule for the web page in the head section
  • -The rules of internal would only effect the classes and ids of that page alone
  • EXTERNAL- most prefered method b/c it allows devs to keep CSS rules in a seperate file
  • -Using the * command in our style sheet we are Declaring that everthing on the page will have the style attchached to its command
